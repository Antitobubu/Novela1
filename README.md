# Novela1

Novela1 es un proyecto de novela visual en etapa inicial. La idea es construir una primera demo corta y jugable que presente el tono de la historia, sus personajes principales y decisiones que tengan consecuencias visibles para el jugador.

## Objetivo del proyecto

Crear una novela visual enfocada en narrativa, decisiones y atmosfera. La primera meta es desarrollar una demo de 15 a 30 minutos que permita probar la historia antes de expandirla.

## Alcance inicial

- Una ruta principal.
- Dos decisiones importantes.
- Dos finales o consecuencias distintas.
- Tres a cinco personajes principales.
- Tres a cinco escenarios iniciales.
- Musica y efectos basicos para reforzar el tono.

## Tecnologia sugerida

El motor recomendado para comenzar es **Ren'Py**, porque esta disenado especialmente para novelas visuales y permite trabajar rapido con dialogos, escenas, imagenes, musica, decisiones y finales.
